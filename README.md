# Sardor Converter Bot

Telegram bot - hujjatlarni konvertatsiya qiladi.

## Imkoniyatlar
- DOCX → PDF
- PDF → DOCX
- PPTX → PDF/DOCX

## 24/7 Online qilish

### Render.com (BEPUL)

1. GitHub'ga yuklash:
```bash
git init
git add .
git commit -m "Bot tayyor"
git remote add origin https://github.com/SIZNING_USERNAME/sardor-bot.git
git push -u origin main
```

2. https://render.com ga kiring (GitHub bilan)

3. "New +" → "Web Service"

4. GitHub repository'ni ulang

5. Sozlamalar:
   - Name: sardor-converter-bot
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python bot.py`
   - Instance Type: Free

6. "Create Web Service" bosing

7. Tayyor! Bot 24/7 ishlaydi.
